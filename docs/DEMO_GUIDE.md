# 🎬 项目演示指南

## 📹 演示视频制作指南

本文档提供制作项目演示视频的详细指南，帮助您展示项目的所有亮点。

## 🎯 视频目标

演示视频应该展示：
1. **项目的核心价值** - 隐私保护的薪资管理
2. **技术亮点** - FHE 加密技术的应用
3. **完整功能** - 所有主要功能模块
4. **用户体验** - 直观的界面和流畅的操作

## 📋 视频大纲（建议时长：5-8 分钟）

### 第一部分：项目介绍（30 秒）
- 项目名称和定位
- 核心价值主张
- 技术栈简介

### 第二部分：技术架构（1 分钟）
- 系统架构图展示
- FHE 加密技术说明
- 区块链集成说明

### 第三部分：功能演示（4-5 分钟）

#### 3.1 部门管理（30 秒）
- 创建部门
- 设置加密预算
- 查看部门列表

#### 3.2 员工管理（30 秒）
- 添加员工
- 分配角色和部门
- 员工列表展示

#### 3.3 薪资管理（1 分钟）
- 提交加密薪资
- 查看加密薪资记录
- 员工查看自己的薪资（自动解密）
- **重点展示**：加密过程可视化

#### 3.4 统计分析（1 分钟）
- 加密数据统计（不解密）
- 部门总薪资计算
- 平均薪资计算
- 统计图表展示
- **重点展示**：在不解密的情况下进行计算

#### 3.5 权限管理（30 秒）
- 角色权限矩阵
- 权限分配
- 权限说明

#### 3.6 技术特性（1 分钟）
- FHE 加密过程可视化
- 安全证明展示
- 合规性说明（GDPR、CCPA）
- 系统架构图

### 第四部分：总结（30 秒）
- 项目亮点总结
- 技术优势
- 应用场景

## 🎥 录制步骤

### 准备工作

1. **环境准备**
   - 确保本地开发服务器运行正常
   - 准备好演示数据（使用一键生成演示数据功能）
   - 检查所有功能正常工作

2. **录制工具**
   - 推荐使用 OBS Studio（免费，功能强大）
   - 或使用系统自带录屏工具
   - 确保录制分辨率：1920x1080

3. **浏览器设置**
   - 使用 Chrome 或 Edge（最新版本）
   - 清除浏览器缓存
   - 使用无痕模式（避免扩展干扰）

### 录制流程

1. **开始录制**
   - 打开浏览器，访问 `http://localhost:3000`
   - 连接钱包（MetaMask）
   - 切换到 Sepolia 测试网

2. **功能演示**
   - 按照视频大纲顺序演示
   - 每个功能停留 3-5 秒
   - 重要操作可以慢速展示

3. **结束录制**
   - 展示项目 GitHub 链接
   - 展示在线演示链接（如果有）

## ✂️ 后期制作建议

### 视频编辑

1. **添加标题和字幕**
   - 每个功能模块添加标题
   - 重要操作添加说明文字
   - 使用清晰的字体和颜色

2. **添加背景音乐**
   - 选择轻松、专业的背景音乐
   - 音量控制在 20-30%
   - 确保不干扰旁白（如果有）

3. **添加转场效果**
   - 使用简单的淡入淡出
   - 避免过于花哨的效果

4. **添加标注**
   - 重要功能添加箭头或高亮
   - 关键数据添加标注

### 视频优化

- **分辨率**: 1920x1080 (1080p)
- **帧率**: 30fps 或 60fps
- **格式**: MP4 (H.264)
- **时长**: 5-8 分钟
- **文件大小**: < 100MB（如果上传到 YouTube）

---

## 📝 无旁白版本

如果您想制作无旁白版本（纯操作演示），建议：

1. **添加字幕说明**
   - 每个操作添加简短文字说明
   - 使用清晰的字体和颜色
   - 字幕停留时间 2-3 秒

2. **添加背景音乐**
   - 选择轻松、专业的背景音乐
   - 音量控制在 30-40%

3. **加快播放速度**
   - 可以适当加快（1.2x-1.5x）
   - 但确保操作清晰可见
