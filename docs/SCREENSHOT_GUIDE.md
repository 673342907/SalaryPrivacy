## 🛠️ 截图工具推荐

### Windows 用户

#### 方法 1: Windows 自带截图工具（最简单）
1. 按 `Win + Shift + S` 打开截图工具
2. 选择"矩形截图"或"全屏截图"
3. 选择要截图的区域
4. 截图会自动保存到剪贴板
5. 打开画图工具（Win + R，输入 `mspaint`）
6. 粘贴（Ctrl + V）
7. 保存为 PNG 格式到 `docs/images/screenshots/` 目录

#### 方法 2: Snipping Tool（截图工具）
1. 按 `Win` 键，搜索"截图工具"
2. 打开截图工具
3. 点击"新建"
4. 选择要截图的区域
5. 点击"保存"按钮
6. 保存到 `docs/images/screenshots/` 目录

#### 方法 3: 浏览器扩展（推荐）
- **Awesome Screenshot** (Chrome/Edge)
  - 安装: Chrome Web Store 搜索 "Awesome Screenshot"
  - 使用: 点击扩展图标 → "Capture visible part of page" 或 "Capture entire page"
  - 保存: 下载到 `docs/images/screenshots/` 目录

- **Nimbus Screenshot** (Chrome/Edge)
  - 安装: Chrome Web Store 搜索 "Nimbus Screenshot"
  - 使用: 点击扩展图标 → 选择截图类型
  - 保存: 下载到 `docs/images/screenshots/` 目录

### 专业工具（可选）

- **ShareX** (免费，功能强大)
  - 下载: https://getsharex.com/
  - 功能: 截图、编辑、上传、OCR
  - 推荐用于批量截图

- **Greenshot** (免费，开源)
  - 下载: https://getgreenshot.org/
  - 功能: 截图、编辑、标注
  - 轻量级，易于使用

## 📐 截图要求

### 分辨率
- **最小**: 1920x1080 (1080p)
- **推荐**: 2560x1440 (1440p) 或更高
- **注意**: 如果浏览器窗口较小，可以放大到 125% 或 150%

### 格式
- **推荐**: PNG（无损压缩，质量最好）
- **备选**: JPG（文件更小，但可能有压缩）

### 文件大小
- **目标**: 每个文件 < 2MB
- **如果太大**: 可以使用图片压缩工具（如 TinyPNG）

### 命名规范
- 使用小写字母
- 使用连字符（-）分隔单词
- 示例: `dashboard.png`, `department-management.png`

## 🎨 截图技巧

### 1. 准备页面
- 确保页面完全加载
- 关闭不必要的浏览器扩展提示
- 隐藏敏感信息（如真实钱包地址）
- 使用演示数据（如果有）

### 2. 选择时机
- 等待所有动画完成
- 等待数据加载完成
- 等待图表渲染完成
- 确保没有错误提示

### 3. 截图区域
- 截取整个页面（推荐使用浏览器扩展的"整页截图"功能）
- 如果页面太长，可以截取关键部分
- 确保重要元素都在截图内

### 4. 后期处理（可选）
- 使用图片编辑工具添加标注
- 添加箭头指向重要功能
- 添加文字说明（如果需要）
- 调整亮度和对比度（如果需要）

## 🚀 快速开始

1. **启动应用**
   ```bash
   cd packages/nextjs
   pnpm dev
   ```

2. **打开浏览器**
   - 访问 `http://localhost:3000/confidential-salary`
   - 或访问在线演示地址

3. **准备数据**
   - 使用"一键生成演示数据"功能
   - 或手动创建一些测试数据

4. **开始截图**
   - 使用推荐的截图工具
   - 按照本指南逐个截图
   - 保存到 `docs/images/screenshots/` 目录

5. **更新 README**
   - 截图会自动显示在 README 中
   - 如果截图路径正确，无需额外操作

## 💡 提示

- **批量截图**: 可以一次性截图所有页面，然后逐个保存
- **截图顺序**: 建议按照页面导航顺序截图
- **数据准备**: 确保有足够的演示数据，让截图更丰富
- **浏览器**: 建议使用 Chrome 或 Edge，界面更现代
- **分辨率**: 如果显示器分辨率较低，可以临时调整浏览器缩放
