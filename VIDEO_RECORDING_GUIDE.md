## 🛠️ 推荐录制工具（简单易用）

### 方案 1: OBS Studio（免费，推荐）⭐⭐⭐⭐⭐

**优点：**
- 完全免费
- 功能强大
- 支持多种录制格式
- 可以录制屏幕 + 摄像头

**下载：** https://obsproject.com/

**快速设置：**
1. 下载并安装 OBS Studio
2. 打开 OBS
3. 点击"场景" → "添加" → "显示器采集"
4. 点击"开始录制"
5. 完成录制后点击"停止录制"

### 方案 2: Windows 自带录制（最简单）⭐⭐⭐⭐⭐

**Windows 10/11 自带功能：**

**快捷键：** `Win + G` 打开 Xbox Game Bar

**步骤：**
1. 按 `Win + G` 打开录制工具栏
2. 点击录制按钮（或按 `Win + Alt + R`）
3. 开始演示
4. 再次按 `Win + Alt + R` 停止录制
5. 视频保存在：`C:\Users\你的用户名\Videos\Captures`

**优点：**
- 无需安装
- 操作简单
- 一键录制

### 方案 3: 在线工具（无需安装）⭐⭐⭐⭐

**推荐工具：**
- **Loom** (https://www.loom.com/) - 免费，简单
- **Screencastify** (Chrome 扩展) - 简单易用
- **Kap** (Mac) - 开源免费

## 🎬 录制步骤（超简单）

### 准备工作（5 分钟）

1. **打开项目**
   ```bash
   # 确保开发服务器运行
   cd packages/nextjs
   pnpm dev
   ```

2. **打开浏览器**
   - 访问 http://localhost:3000
   - 确保所有功能正常

3. **准备脚本**
   - 打开 `VIDEO_SCRIPT_NO_VOICE.md`
   - 放在另一个屏幕上（或打印出来）

4. **打开录制工具**
   - 使用 Windows 自带：按 `Win + G`
   - 或打开 OBS Studio

### 录制流程（10 分钟）

#### 第 2 部分：功能演示（5-8 分钟）

**按照 `VIDEO_SCRIPT_NO_VOICE.md` 的步骤：**

1. **首页展示**（30 秒）
   - 展示项目首页
   - 展示功能导航

2. **Dashboard 演示**（1 分钟）

   - 创建部门
