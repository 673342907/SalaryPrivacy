## 1. 准备工作

### 1.1 确保应用可以访问
1. 打开浏览器，访问你的 Vercel 部署地址
   - 例如：`https://salary-privacy-xxx.vercel.app`
   - 或者本地运行：`http://localhost:3000`

2. 确保页面可以正常加载
   - 检查所有功能是否正常
   - 确保没有错误提示

### 1.2 准备钱包
1. 打开 MetaMask 浏览器扩展
2. 确保钱包中有 Sepolia 测试网 ETH
   - 如果没有，可以从 [Sepolia Faucet](https://sepoliafaucet.com/) 获取
3. 切换到 Sepolia 测试网
   - 在 MetaMask 中选择网络 → Sepolia

## 2. 安装录制软件

### 方法一：使用 OBS Studio（推荐，免费）

#### 步骤 1：下载安装
1. 访问 https://obsproject.com/
2. 点击 "Download" 按钮
3. 选择 Windows 版本下载
4. 运行安装程序，按照提示安装

#### 步骤 2：基本设置
1. 打开 OBS Studio
2. 首次打开会看到设置向导，选择：
   - "Optimize for streaming" 或 "Optimize for recording"
   - 选择 "I will only be recording"
3. 点击 "Next" 完成设置

#### 步骤 3：创建场景
1. 在左下角 "Scenes" 区域，点击 "+" 按钮
2. 输入场景名称：`ConfidentialSalary Demo`
3. 点击 "OK"

#### 步骤 4：添加浏览器窗口
1. 在 "Sources" 区域，点击 "+" 按钮
2. 选择 "Window Capture"（窗口捕获）
3. 输入名称：`Browser Window`
4. 点击 "OK"
5. 在 "Window" 下拉菜单中选择你的浏览器窗口
6. 点击 "OK"

#### 步骤 5：调整画面大小
1. 在预览窗口中，拖动红色边框调整大小
2. 确保浏览器窗口完全显示
3. 可以右键点击 → "Transform" → "Fit to Screen" 自动适配

#### 步骤 6：设置录制参数
1. 点击右下角 "Settings" 按钮
2. 选择 "Output"（输出）
3. 设置：
   - Recording Path: 选择一个文件夹保存视频
   - Recording Format: MP4
   - Encoder: x264
   - Video Bitrate: 5000 Kbps
4. 点击 "OK"

### 方法二：使用 Windows 内置录制（简单但功能有限）

#### 步骤 1：打开录制工具
1. 按 `Win + G` 键打开 Xbox Game Bar
2. 如果没有，可以在 Microsoft Store 搜索 "Xbox Game Bar" 安装

#### 步骤 2：开始录制
1. 点击录制按钮（红色圆圈）
2. 或者按 `Win + Alt + R` 开始录制
3. 再次按 `Win + Alt + R` 停止录制

#### 步骤 3：查看视频
- 视频保存在：`C:\Users\你的用户名\Videos\Captures`

## 4. 开始录制

### 4.1 录制前最后检查

#### 检查清单：
- [ ] 浏览器窗口已全屏（F11）
- [ ] 钱包已连接
- [ ] 演示数据已生成
- [ ] OBS Studio 已打开并设置好
- [ ] 录制路径已设置
- [ ] 浏览器中没有其他标签页干扰

### 4.2 开始录制

#### 步骤 1：在 OBS Studio 中开始录制
1. 点击右下角 "Start Recording"（开始录制）按钮
2. 或者按 `Ctrl + R` 快捷键
3. 看到红色 "Recording" 指示器表示正在录制

#### 步骤 2：按照脚本操作

打开 `VIDEO_SCRIPT_NO_VOICE.md` 文件，按照以下顺序操作：

**第 1 部分：开场（20秒）**
1. 在浏览器中展示首页
2. 鼠标指向项目标题
3. 缓慢滚动，展示核心功能卡片
4. 暂停 2-3 秒让观众看清楚

**第 2 部分：系统概览（40秒）**
1. 滚动到技术架构部分
2. 鼠标指向架构图
3. 展示各个组件（前端、FHEVM、智能合约、区块链）
4. 可以添加文字说明（后期制作时）

**第 3 部分：钱包连接（30秒）**
1. 点击右上角 "连接钱包" 按钮
2. 选择 MetaMask
3. 在 MetaMask 弹窗中确认
4. 等待连接完成
5. 展示 FHEVM 状态（显示 "就绪"）

**第 4 部分：一键生成演示数据（1分钟）**
1. 滚动到 Dashboard 页面
2. 找到 "一键生成演示数据" 按钮
3. 点击按钮（鼠标停留 1-2 秒）
4. 等待加载动画完成
5. 展示成功提示（显示数据统计）

**第 5 部分：部门管理（50秒）**
1. 点击顶部导航栏的 "部门管理" 标签
2. 展示部门列表
3. 鼠标指向一个部门卡片
4. 点击 "查看详情" 按钮
5. 展示部门信息（预算、员工数等）
6. 强调 "加密存储" 文字

**第 6 部分：员工管理（50秒）**
1. 点击 "员工管理" 标签
2. 展示员工列表
3. 鼠标指向不同角色的员工
4. 展示角色标签（Admin、HR、Manager、Employee）
5. 可以点击 "编辑" 按钮展示功能（虽然只是提示）

**第 7 部分：薪资管理（1分20秒）**
1. 点击 "薪资管理" 标签
2. 展示薪资列表
3. 点击 "提交加密薪资" 按钮
4. 输入员工地址（可以输入演示数据中的地址）
5. 输入金额（例如：30000）
6. 点击 "提交" 按钮
7. 等待加密动画完成
8. 展示成功提示
9. 展示薪资列表中的新记录

**第 8 部分：统计分析（1分30秒）**
1. 点击 "统计分析" 标签
2. 展示 FHE 计算演示组件
3. 点击 "➕ 加密加法" 按钮
4. 等待计算完成
5. 展示结果（强调"不解密原始数据"）
6. 点击 "📊 加密平均值" 按钮
7. 展示计算结果
8. 展示统计图表
9. 鼠标指向图表，展示数据

**第 9 部分：权限管理（30秒）**
1. 点击 "权限管理" 标签
2. 展示角色权限表格
3. 鼠标指向不同角色
4. 展示权限说明

**第 10 部分：技术亮点（1分钟）**
1. 滚动到 Dashboard 底部的技术亮点部分
2. 展示 FHE 加密可视化
3. 展示安全证明
4. 展示合规性说明
5. 展示 Zama 集成

**第 11 部分：总结（30秒）**
1. 回到首页
2. 展示项目信息
3. 可以展示 GitHub 链接（如果有）

#### 步骤 3：停止录制
1. 在 OBS Studio 中点击 "Stop Recording"（停止录制）
2. 或者按 `Ctrl + R` 快捷键
3. 视频会自动保存到设置的路径

## 5. 后期制作

### 5.1 安装视频编辑软件

#### 推荐：DaVinci Resolve（免费，功能强大）

1. 访问 https://www.blackmagicdesign.com/products/davinciresolve
2. 下载 DaVinci Resolve（免费版）
3. 安装软件

### 5.2 添加文字说明

#### 步骤 1：导入视频
1. 打开 DaVinci Resolve
2. 创建新项目：`ConfidentialSalary Demo`
3. 在 "Media Pool" 中右键 → "Import Media"
4. 选择你录制的视频文件

#### 步骤 2：添加到时间线
1. 将视频拖拽到时间线（Timeline）
2. 视频会显示在时间线上

#### 步骤 3：添加文字说明

**方法一：使用标题功能**
1. 点击顶部 "Effects"（效果）标签
2. 在搜索框输入 "Text"
3. 找到 "Text+" 效果
4. 拖拽到时间线上（放在视频轨道上方）
5. 在右侧 "Inspector" 面板中：
   - 输入文字内容（从 `VIDEO_SCRIPT_NO_VOICE.md` 复制）
   - 调整字体大小（建议 24-32）
   - 调整位置（建议底部居中）
   - 设置背景色（半透明黑色）
   - 设置文字颜色（白色）

**方法二：使用字幕功能**
1. 点击顶部 "Edit"（编辑）标签
2. 在时间线下方找到 "Subtitles"（字幕）轨道
3. 右键 → "Add Subtitle"
4. 输入文字内容
5. 调整时间位置

#### 步骤 4：添加章节标题

在每个功能模块开始处添加章节标题：

1. 在时间线上找到功能切换的位置
2. 添加 "Text+" 效果
3. 输入章节标题，例如：
   - "部门管理"
   - "员工管理"
   - "薪资管理"
   - "统计分析"
4. 设置大字体（48-64）
5. 设置居中显示
6. 设置显示时间（3-5秒）

#### 步骤 5：添加关键操作高亮

对于重要的按钮点击，可以添加圆圈高亮：

1. 在按钮点击的位置，添加 "Text+" 效果
2. 输入 "○" 或使用图形
3. 调整大小和位置，覆盖在按钮上
4. 设置动画效果（淡入淡出）

### 5.3 添加背景音乐（可选）

1. 准备背景音乐文件（建议使用无版权音乐）
2. 在 "Media Pool" 中导入音乐
3. 拖拽到时间线的音频轨道
4. 调整音量（建议 -20dB 到 -30dB，不要太大）
5. 确保音乐不会干扰理解

### 5.4 导出视频

1. 点击顶部 "Deliver"（交付）标签
2. 设置导出参数：
   - Format: MP4
   - Codec: H.264
   - Resolution: 1920x1080
   - Frame Rate: 30 fps
   - Quality: High
3. 选择保存位置
4. 点击 "Add to Render Queue"（添加到渲染队列）
5. 点击 "Render All"（渲染全部）
6. 等待导出完成

## 🆘 常见问题

### Q1: OBS Studio 录制的视频没有声音？
**A**: 无旁白视频不需要声音，这是正常的。如果需要背景音乐，在后期制作时添加。

### Q2: 录制时画面卡顿？
**A**: 
- 降低录制分辨率（720p）
- 降低帧率（30fps）
- 关闭其他占用资源的程序

### Q3: 如何添加鼠标点击效果？
**A**: 
- 在 OBS Studio 中安装 "Mouse Highlight" 插件
- 或者在后期制作时添加圆圈动画

### Q4: 视频文件太大？
**A**: 
- 在导出时降低码率
- 或者使用视频压缩工具（如 HandBrake）

### Q5: 不知道如何添加文字？
**A**: 
- 可以使用更简单的工具：Kapwing（在线编辑器）
- 访问 https://www.kapwing.com/
- 上传视频，添加文字，导出

10. 下载并上传到 YouTube
