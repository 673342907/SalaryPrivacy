# 🎬 5 分钟快速录制指南

## 🚀 最简单的方法（推荐）

### 使用 Windows 自带录制工具

#### 步骤 1: 准备（1 分钟）

1. 打开项目：
   ```bash
   cd packages/nextjs
   pnpm dev
   ```

2. 打开浏览器：http://localhost:3000

3. 打开脚本：`VIDEO_SCRIPT_NO_VOICE.md`

#### 步骤 2: 开始录制（1 分钟）

1. 按 `Win + G` 打开录制工具栏
2. 点击录制按钮（红色圆点）
3. 或按快捷键：`Win + Alt + R`

#### 步骤 3: 演示（8 分钟）

**按照这个顺序操作（不需要说话）：**

1. **展示首页**（30 秒）
   - 打开 http://localhost:3000
   - 展示功能导航

2. **生成演示数据**（30 秒）
   - 进入薪资管理页面
   - 点击"一键生成演示数据"
   - 等待数据生成

3. **展示部门管理**（1 分钟）
   - 切换到"部门管理"标签
   - 展示部门列表
   - 点击"创建部门"
   - 展示区块链模式切换

4. **展示员工管理**（1 分钟）
   - 切换到"员工管理"标签
   - 展示员工列表
   - 点击"添加员工"
   - 填写表单并提交

5. **展示薪资管理**（2 分钟）
   - 切换到"薪资管理"标签
   - 点击"提交薪资"
   - 填写并提交（展示加密过程）
   - 点击"查看薪资"（展示解密过程）

6. **展示统计分析**（1 分钟）
   - 切换到"统计分析"标签
   - 展示加密计算功能
   - 展示预算合规检查

7. **展示自动化脚本**（1 分钟）
   - 打开终端
   - 运行：`cd packages/hardhat && npx ts-node scripts/generate-docs.ts`
   - 展示生成的文档

8. **总结**（30 秒）
   - 回到 GitHub 仓库
   - 展示项目结构
   - 结束

#### 步骤 4: 停止录制（10 秒）

1. 按 `Win + Alt + R` 停止录制
2. 视频自动保存到：`C:\Users\你的用户名\Videos\Captures`

#### 步骤 5: 上传（2 分钟）

1. 找到视频文件
2. 上传到 YouTube（推荐）或 Vimeo
3. 设置为"不公开"或"公开"
4. 复制视频链接

## 🎯 如果遇到问题

### 问题：找不到录制工具
**解决：** 按 `Win + G`，如果没有，使用 OBS Studio

### 问题：录制质量不好
**解决：** 
- 确保屏幕分辨率足够
- 关闭不必要的程序
- 使用全屏模式

### 问题：操作出错
**解决：** 
- 暂停录制
- 修复问题
- 继续录制
- 或重新录制该部分
