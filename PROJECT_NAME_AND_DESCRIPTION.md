## 📝 项目描述

### 简短描述（一句话）

**ConfidentialPay** 是一个基于全同态加密（FHE）技术的去中心化隐私保护薪资管理系统，确保薪资数据在区块链上加密存储，只有授权用户才能解密查看。

### 详细描述

**ConfidentialPay** 是一个创新的去中心化应用（dApp），利用 Zama 的 FHEVM（全同态加密虚拟机）技术，实现了完全隐私保护的薪资管理系统。

#### 核心特性

1. **🔐 全同态加密保护**
   - 所有薪资数据在链上以加密形式存储
   - 支持在不解密的情况下进行加密数据计算
   - 只有授权用户才能解密查看原始数据

2. **👥 基于角色的权限管理**
   - **Admin（管理员）**: 完全权限，可以管理所有功能和分配角色
   - **HR（人力资源）**: 可以创建部门、管理员工、提交薪资
   - **Manager（经理）**: 可以查看部门数据和员工薪资
   - **Employee（员工）**: 只能查看自己的薪资

3. **🏢 完整的组织管理**
   - 部门创建和管理
   - 员工添加和角色分配
   - 加密预算设置和合规检查

4. **💰 隐私保护的薪资管理**
   - HR/Admin 可以提交加密薪资
   - 员工可以查看自己的薪资（自动解密）
   - 支持加密统计和预算合规检查

5. **🔒 零知识保证**
   - 智能合约无法看到原始薪资数据
   - 所有操作都在链上可验证
   - 完全透明且不可篡改

#### 技术栈

- **区块链**: Ethereum Sepolia 测试网
- **加密技术**: Zama FHEVM（全同态加密虚拟机）
- **前端框架**: React + TypeScript
- **智能合约**: Solidity (0.8.24)
- **开发工具**: Hardhat
- **UI框架**: Tailwind CSS

#### 应用场景

- 企业内部薪资管理
- 去中心化组织（DAO）的薪酬系统
- 需要隐私保护的薪资审计
- 多部门企业的薪资统计

#### 技术亮点

1. **FHE 加密计算**: 在不解密的情况下进行薪资统计和预算比较
2. **权限控制**: 基于智能合约的角色权限系统
3. **用户友好**: 直观的 Web 界面，易于使用
4. **安全可靠**: 所有数据加密存储，权限链上验证
