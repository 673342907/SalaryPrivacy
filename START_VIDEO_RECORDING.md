# 🎬 一键启动视频录制

## 🚀 最简单的方法

### 步骤 1: 打开录制助手（10 秒）

1. 确保开发服务器运行：
   ```bash
   cd packages/nextjs
   pnpm dev
   ```

2. 在浏览器中打开：
   ```
   http://localhost:3000/video-demo-helper.html
   ```

### 步骤 2: 开始录制（5 秒）

1. 点击"开始录制"按钮
2. 按 `Win + G` 打开 Windows 录制工具
3. 点击录制按钮（红色圆点）

### 步骤 3: 按照助手操作（10 分钟）

1. 助手会显示当前步骤
2. 按照步骤内容操作
3. 完成一步后，点击"下一步"
4. 助手会自动跳转到下一步

### 步骤 4: 完成录制（10 秒）

1. 所有步骤完成后，停止录制（`Win + Alt + R`）
2. 视频自动保存
3. 上传到 YouTube 或 Vimeo

## 🎯 使用技巧

### 1. 保持助手页面打开
- 在另一个窗口或屏幕上打开
- 随时查看当前步骤

### 2. 可以暂停
- 如果出错，点击"暂停"
- 修复问题后继续

### 3. 可以重置
- 如果想重新开始，点击"重置"

### 4. 按照步骤操作
- 不要跳过步骤
- 每个步骤都很重要
